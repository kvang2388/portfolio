🚀 Featured Project
🎉 Happenings – Social Event Planner

A full-stack social event platform designed to allow users to create, browse, and receive personalized event recommendations based on user behavior and preferences.

Key Features:

Secure user authentication with JWT and password encryption (BCrypt)
Create, view, and manage events stored in a relational database
Category-based event recommendation system using saved user preferences
Real-time frontend and backend integration using REST APIs
Fully database-driven system with no mock or static data
End-to-end workflow: login → events → save events → recommendations

System Architecture:

Backend: Java Spring Boot (REST API)
Frontend: HTML, CSS, JavaScript (fetch API integration)
Database: MySQL (relational schema with foreign keys)
ORM: Hibernate / JPA
Deployment: Render (Dockerized Spring Boot application)

Database Design:

Users table (authentication + profile data)
Events table (event details + ownership)
Categories table (event classification)
Saved Events table (user preferences tracking)

My Role:

Full integration and testing lead
Built event system APIs and ensured production stability
Implemented recommendation logic based on user activity
Verified database integrity and end-to-end system flow
